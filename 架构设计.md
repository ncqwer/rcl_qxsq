# 架构设计

3个模块，顶部用于切分路由，

## 路由设计

- 历史数据统计（/datav）
	- 降水数据库（/precipitation）
	- 积水数据库（/ponding）
	- 水道河情数据库（/watercourse）
	- 降水-积水数据库（/relationBPP）
- 阈值参数展示（/prediction）
- 预报预警程序（/warning）

## 数据库

每个数据库均由一到多个图标呈现。


## 评分细则的设计

```ts
type Rule = {
	min: number;
	max: number;
	score: number;
}

type VariableNames = 'X' | 'Y' | 'Z' | 'T' | 'H';

type ScoreSheet = Record<VariableNames,{
	weight: number;
	rules: Rule[]
}>;
```
